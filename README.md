# 💫 About Me

> *Final-year Computer Science student passionate about AI, software engineering, and building things that solve real problems.*

Hi, I'm Abdul Khader.

I'm a Computer Science student who enjoys learning by building. My journey into technology started with curiosity about how software works, and over time that curiosity grew into a strong interest in Artificial Intelligence, Full Stack Development, and modern software systems.

Most of my learning happens through projects. Whether it's developing AI-powered applications, experimenting with Large Language Models, building RAG systems, working on computer vision projects, or creating full-stack web applications, I enjoy turning ideas into something people can actually use.

I'm currently exploring how AI can move beyond simple chat experiences and become a practical tool that helps users solve real-world problems. At the same time, I'm continuously improving my skills in backend development, system design, and scalable application development.

I don't claim to know everything—I'm still learning, experimenting, making mistakes, and growing with every project I build.
